Arm Allinea Studio/Arm Forge trials package information
-------------------------------------------------------

The Arm Allinea Studio/Arm Forge Trials package consists of the following directories:

* /src/: Tutorial source code files in C, F90, and Python, with Makefiles.
* /doc/: Documentation in HTML and PDF formats. To navigate the HTML documentation, in a web browser open: /html/index.html.
* /license_terms/: License agreement, redistributables statement, supplementary terms statement, and third party license statement.
