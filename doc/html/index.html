Arm Allinea Studio/Arm® Forge Trials Package Documentation

Welcome to the documentation for the Arm Allinea Studio/Arm® Forge trials package.

Thank you for requesting a trial license of the Arm HPC tools. The Arm HPC tools suite consists of the following components:

  • Arm Fortran Compiler for compiling on Arm-based systems.
  • Arm C/C++ Compiler for compiling on Arm-based systems.
  • Arm Performance Libraries for optimizing on Arm-based systems.
  • Arm Forge for debugging and profiling on any HPC architecture.
  • Arm Performance Reports for analyzing performance on any HPC architecture.

An Arm Allinea Studio trial license allows you to evaluate all of the features of the tools listed above.

An Arm Forge trial license allows you to evaluate all of the features of the following tools:

  • Arm Forge
  • Arm Performance Reports

This package contains a tutorial with example codes in C/C++, Fortran, and Python that demonstrates:

  • How to fix crashes in parallel applications.
  • How to identify performance issues and optimize them.
